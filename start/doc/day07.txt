# day07

배열
	배열의 데이터를 기억시키는 방법
	배열변수[위치값] = 데이터; 여기서 위치값은 0부터
	
	배열의 내용을 확인하는 방법
	원칙은 하나씩 꺼내서 확인하는것
	내용확인 용도로만 사용할 것이라면
		Arrays.toString(배열변수); <-를 쓸 수 있다. 절대 확인용도로만 사용해야 한다
------------
배열정리
	같은형태의 데이터를 여러개 모아서 한꺼번에 관리하는 방식
	데이터가 많은경우 원칙적으로 데이터 갯수만큼 변수를 만들어야 하지만
	만들때도 불편, 사용할 때도 불편
	
	이런 문제를 해결하기 위해 제시된 방법으로 
	하나의 변수를 이용하여 여러개의 데이터를 관리할 수 있도록 하는 것
자바의 변수형태
	1. 기본형 변수형태(Primitive type 또는 value 형태)
		=>자바에서 제공되는 기본 데이터 타입으로 만드는 변수
		boolean char byte short int ling float double 8종
	2. 참조형 변수(Reference Type)
		=>기본 데이터 타입 이외의 모든 변수
------------
Heap영역
	특징
		1.반드시 주소를 알아야 사용할 수 있는 영역
		2.메모리의 8가지 영역 중 크기가 가장 큰 영역, 또한 확장도 가능하다.
		(부족할 시 스스로가 다른 영역을 빼앗아서 Heap영역으로 사용 할 수 있다.
		따라서 자바에서는 많은 양의 데이터를 필요로 하면 이것은 Heap 영역에 기억하도록 되어있다.
------
배열 만드는 방법
	1.배열은 원칙적으로 데이터가 기억될 공간을 Heap 에 만들어야 한다.
	명령
		new 데이터타입[];
			
			데이터타입:Heap에 만든 데이터 영역의 형태를 지정(관리할 데이터들의 타입)
			갯수	  :Heap에 만들 데이터 영역의 갯수 지정(관리할 데이터가 몇개인지)
	예
		new int[10];
		=> Heap영역에 int 데이터들을 기억할 메모리 공간을 10 확보하세요(아직 데이터는 없고 0으로 채워짐)
	참고
		new명령
			1.Heap에 먼가를 만들겠습니다. 라는 뜻
			2.주소는 알려드리겠습니다 라는 뜻
	
	2. Heap에 있는 메모리의 주소를 기억할 변수를 만들어야 한다.
	 명령
	 	데이터타입[] 변수이름;
	 참고
	 	데이터타입[]
	 		자신이 관리할 Heap영역에 있는 실제 데이터의 형태에 따라서 달라진다 -> Heap Type라고함
	 주의
	 	Heap Tpye 이 같은 주소만 기억한다
	 	
	 예 ]
	 	int[] num;  <- 정수 여러개를 관리할 배열을 기억하기 위한 변수만을 선언하는 것
	 주의 ]
	 	int[] num = new float[5]; <- 잘못됨
	 	int[] num = new int[5];   <- 잘 됨
	 저리 ]
	 	1.만들 배열이 Heap에 생길 때 그 주소를 기억할 변수를 준비
	 	2.Heap에 필요한 메모리를 만들고 그 주소를 준비한 변수에 기억시킨다.
-------
배열 사용방법
	명령어]
		변수[위치값];
		
	참고 ]
		-위치값은 0부터 시작
		-배열주소.(변수)가 관리하는 곳에 있는 메모리의 갯수를 알려주는 명령
----
참고 ]
	배열의 초기화
	=> 배열 메모리는  Heap에 생기므로 자동으로 초기화가 이루어진다.
	정수는 0, 실수는 0.0, 논리는 false, 문자는 ASCII 0, 경우에 따라서 배열 메모리에 강제로 특정값을 입력할 필요고 있다.
	이것을 배열의 초기화라고 한다.
	형식1.
		int[] num = {1,2,3,4,5};			<-둘 다 맞는 표현식
	형식2.
		int[] num = new int[]{1,2,3,4,5};	<-둘 다 맞는 표현식
	형식3(내가 추가한것)
		int[] num;
		num = new int[5]
		num[0] = 1
			...
		num[4] = 5
	형식4(내가추가한것)
		int[] num = new int[5]
		num[0] = 1
			...
		num[4] = 5
		
			
	주의]
		배열을 초기화 할 경우에는 배열의 크기는 입력하면 안된다. 대신 초기화 데이터를 이용해서 자동으로 배열의 크기가 결정
	참고]
		Arrays.toString() 함수
		배열을 출력하기 위해서는 반복명령을 이용해서 출력하는것이 원칙.
		하지만 단순히 배열의 내용만 확인하는 차원에서는 for반복문들 이용하는것이 불편
		배열이 이쓴 데이터를 출력하기 위한 명령으로 배열이 가진 모든 데이터를 문자열로 변환해서 반환해주는 기능을 가진 함수
	주의]
		d오직 배열의 내용을 확인하는데에만 써야한다.
